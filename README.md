# 🚗 Vehicle Service Management System (VSMS)

A full-stack web application to manage vehicle service records digitally.
Built with PHP, MySQL, HTML, and CSS.

## 📸 Preview
![VSMS Screenshot](assets/screenshot.png)

## ✨ Features
- 🚗 Supports 2, 3, 4, and 6 Wheeler vehicles
- 🔍 Search services instantly
- 📋 Service request booking system
- 👨‍💼 Admin panel for managing services
- 🧾 Automated invoice generation
- 📊 Customer and vehicle data management

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** XAMPP (Apache)

## ⚙️ How To Run This Project

### Requirements
- XAMPP installed
- PHP 7.4+
- MySQL

### Steps
1. Clone this repository
```bash
git clone https://github.com/sankarmwork-afk/projects.git
```
2. Copy project folder to XAMPP htdocs:
```bash
cp -r vehicle_service /Applications/XAMPP/htdocs/
```
3. Start XAMPP — Apache and MySQL
4. Import database from `database/` folder in phpMyAdmin
5. Open browser and go to:
```
http://localhost/vehicle_service/
```

# C++ and Python Practice Projects

This repository contains my beginner programming projects written in **C++** and **Python**.
These programs were created while learning programming, problem solving, and basic computer science concepts.

## Projects

### 1. Bank Management System (C++)

A simple banking system program that allows users to:

* Create an account
* Deposit money
* Withdraw money
* Display account details

### 2. Age Verification Program (C++)

A simple program that checks whether a person is eligible based on their age.

### 3. Pythagoras Theorem Calculator (C++)

A program that calculates the missing side of a triangle using the Pythagorean theorem.

### 4. Quantum Entanglement Example (C++)

A small experimental program related to quantum concepts.
